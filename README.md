# SYAHRIYAH-VERSI-14
“Web Pembayaran Syahriyah Pondok adalah sistem berbasis web untuk mempermudah pencatatan dan pemantauan pembayaran santri. Sistem ini membantu pengelola pondok mengelola data santri, mencatat transaksi syahriyah, memantau status pembayaran, serta menyajikan laporan secara rapi, cepat, dan transparan.”
